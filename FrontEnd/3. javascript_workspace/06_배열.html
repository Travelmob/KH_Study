배열

자바스크립트에서는 변수 선언 시 별도의 자료형을 지정하지 않기 떄문에
배열도!! var, let, const로 선언을 한다!!!
어떤 자료형이든 상관없이 하나의 배열에 담을 수 있다. (자바의 컬렉션과 유사)

* 배열의 선언

배열 선언 시 배열의 크기를 지정한채로 선언하거나, 따로 크기를 지정하지 않을수도 있음
(배열의 크기의 제약이 없다 => 자바 컬렉션의 장점)

* Array 객체의 메소드

1) indexOf(찾고자하는 요소) : 배열에서 해당 요소가 위치해있느 인덱스를 반환

2) concat(배열, 배열, 배열..) : 여러개의 배열을 결합하고 할 때 사용

3) reverse() : 배열에 담긴 요소들을 역순으로 바꿔주는 메소드

4) sort() : 배열에 담긴 값들을 오름차순으로 정렬해주는 메소드

5_1) push(추가할요소) : 배열의 맨 뒤에 요소를 추가하고 크기도 반환
5_2) pop() : 배열의 맨 뒤 요소를 제거된 요소를 반환

6_1 ) unshift(추가할요소) : 배열의 맨 앞에 요소를 추가하고 크기 반환
6_2 ) shift() : 배열의 맨 앞에 요소를 제거하고 제거된 요소를 반환

7_1) slice(시작인덱스, 끝인덱스) : 배열의 요소들을 추출해주는 메소드
7_2) splice(시작인덱스, 제거수, 추가값) : 배열의 요소를 추출해서 제거 및 추가, 추가값은 생략 가능

8) join(구분자) : 배열에 담긴 값들을 하나의 '문자열'로 합쳐서 반환해주는 메소드
