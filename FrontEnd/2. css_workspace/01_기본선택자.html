선택자란?

특정 HTML요소를 선택하고자 할 떄 사용하는 기능
해당 요소를 선택해서 원하는 style적용 가능

CSS 기본선택자

1. 전체(모든요소) 선택자

        [ 표현법 ]
        * {
            스타일속성 : 값;
            스타일속성 : 값;
        }

        => 모든 요소들 다 선택하고자 할 때 사용

2. 태그 선택자(태그명)

        [ 표현법 ]
        태그명 {
            스타일 속성 : 값;
            스타일 속성 : 값;
        }

        => 이 문서 내에 같은 태그들을 모두 선택할 때 사용

3. 아이디 선택자(#아이디속성값)

        [ 표현법 ]
        #아이디속성값{
            스타일속성 : 값;
            스타일속성 : 값;
            ...
        }
    
        => 이 문서 내에 특정 HTML요소 하나만을 선택하고자 할 때 사용
        단, 해당 요소에 반드시 id속성을 이용해서 고유한 아이디값이 부여되어있어야 함
        같은 태그더라도 부분적으로 다르게 스타일을 부여하고 싶을 떄 사용하면 좋음

  3. 아이디선택자
  4. 아이디선택자

4. 클래스 선택자(.클래스속성값)

        [ 표현법 ]
        .클래스속성값{
            스타일 속성 : 값;
            스타일 속성 : 값;
            ...
        }

        => 이 문서 안에서 여러개의 요소를 선택하고 싶을 때
